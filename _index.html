allong.es

Practical function combinator and decorator recipes from JavaScript Allongé

View on GitHub

allong.es is a JavaScript library based on the function combinator and decorator recipes introduced in the book JavaScript Allongé.

Getting Started

npm install allong.es

Documentation

The decorators and combinators are documented in JavaScript Allongé. There's also a cheat sheet on the README. Some of them have also been described in essays published online. Copies of those essays are included in the project's essays folder.

Project Status

All of the combinators and function decorators from JavaScript Allongé are present. allong.es now also includes functional iterator support.
